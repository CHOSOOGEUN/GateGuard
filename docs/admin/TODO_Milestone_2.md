# 🛰️ GateGuard — Milestone 2.0 (3/31 ~ 4/10) - “High-Intensity Blitz” 목표

> **전략적 지침**: 시험 기간(4월 중순) 동안 개발이 일시 중단될 것을 대비하여, 시스템의 모든 핵심 기능을 “완전 자동화” 및 “통합 완료”하여 서비스가 자가 가동되도록 함. (High Workload & Goal-Oriented)

---

## 🦁 조수근 (백엔드 총괄) 🏗️ [고강도 핵심 작전]

- [ ] **S3 영상 트리밍 & 자동 업로드 파이프라인 완성**: AI 감지 시점 앞뒤 10초 영상 절삭 → 비동기(Celery) S3 업로드 → DB URL 연동 마감 📸
- [ ] **사원번호 기반 통합 인증 서버 완비**: `/api/auth/register` (사원번호 + 비밀번호), `/api/auth/find-pw` 등 핵심 보안 로직 완성 🔐
- [ ] **실시간 알림 디스패처 구축**: WebSocket 외에 Telegram/Email 알림 봇 연동하여 도서관 내에서도 실시간 감지 알림 수신 체계 구축 📡
- [ ] **백엔드 유닛 테스트 커버리지 80% 달성**: 시험 기간 중 서버 오류 방지를 위한 핵심 비즈니스 로직 테스트 코드 작성 🎯

---

## 🏗️ 이동근 · 최태양 (인프라) 🏰 [무인 가동 보장]

- [ ] **Docker Log Rotation & Disk 관리**: 디스크 고갈 방지를 위한 10MB 기준 로그 순환 및 임시 파일 자동 삭제 설정 🐳
- [ ] **PostgreSQL(TimescaleDB) 자동 백업 체계**: S3를 이용한 일일 DB 덤프 자동화 및 복구 스크립트 검증 🗄️🛡️
- [ ] **Nginx Reverse Proxy & SSL 완비**: 모든 통신 채널에 HTTPS 보안 적용 및 도메인 연동 완성 🔐
- [ ] **서버 리소스 자가 모니터링**: CPU/Memory 임계치 초과 시 팀장에게 즉시 알림 발송 설정 📊

---

## 🛢️ 김민지 (DB & PM) 🕵️‍♂️ [데이터 기반 고도화]

- [ ] **통계 View & Materialized View 구축**: 대시보드(ECharts) 로딩 속도 향상을 위한 시간대별/역별 통계 미리 계산 로직 반영 📈
- [ ] **DB Schema 정규화 마감**: 회원가입 정책 변경에 따른 `admins` 테이블 고도화 및 `event_logs` 시계열 파티셔닝 정밀 튜닝 🗄️
- [ ] **High-Density Seed Data 생산**: 다양한 무임승차 시나리오(뒤따르기, 점프, 비상문) 테스트를 위한 대량의 시각화 데이터 삽입 📊

---

## 🧠 윤효정 (AI 파이프라인) 🤖 [이관 과업 및 지능형 고도화]

- [ ] **[M1 이관] AI 코어 기동**: Ultralytics/Supervision 등 라이브러리 구축 및 `ai/inference.py` 첫 구동 성공 🚀
- [ ] **[M1 이관] 트래커 및 블러 안정화**: 객체 ID(Tracker) 유지 및 얼굴 비식별화(Blur) 자동화 파이프라인 기저 완성 🎥
- [ ] **테스트 영상 데이터셋 확보 및 튜닝**: 외부 공개 영상 + 자체 촬영분을 활용한 케이스별 탐지 정확도 정밀 튜닝 (우선 과제) 🎬
- [ ] **동작 인식(Action Recognition) 프로토타입**: 단순 객체 탐지를 넘어 무임승차 '동작'을 구별하는 행동 분석 모델 베타 탑재 🛰️
- [ ] **Line Crossing 좌표 정밀 조정**: 확보된 영상을 기준으로 무임승차 판단을 위한 가상 경계선 좌표 최적화 🎥

- [ ] **다크 모드 & 고해상도 디자인 폴리싱**: 대형 관제 모니터에 최적화된 시인성 높은 프리미엄 UI 마감 🛡️

## 🏁 Milestone 2.0 성공 기준 (Definition of Perfection)

1. [ ] **Self-Sufficient**: 사람이 개입하지 않아도 AI 탐지 -> 영상 저장 -> 통계 반영까지 정지 없이 작동함.
2. [ ] **Integrated**: 프론트엔드의 세련된 UI와 백엔드의 단단한 API가 완전히 하나로 동작함.
3. [ ] **Exam-Proof**: 시험 공부에 전념하는 동안, 서버는 24시간 한 번도 꺼지지 않고 데이터를 수집함.
