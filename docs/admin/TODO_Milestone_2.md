# 🛰️ GateGuard — Milestone 2.0 (3/31 ~ 4/15) - “Blitz Integration” 목표

## 🦁 조수근 (백엔드 총괄) 🏗️

- [ ] **사원번호 기반 인증 시스템 연동**: 사원번호 회원가입(`POST /api/auth/register`) 및 비밀번호 찾기 로직 완성 🔐
- [ ] **타임라인 시각화 API 설계**: 프론트엔드의 “CCTV 기반 타임라인 UI”를 위한 객체 추적 히스토리 데이터 제공 고도화 🛰️
- [ ] **S3 영상 업로드 비동기 마감**: Celery + Redis를 통한 S3 클립 파일 자동 업로드 및 DB URL 매핑 완료 📸
- [ ] **AI 결과 수집 전용 엔드포인트 완비**: 추론 결과를 고속으로 수용하는 `/api/inference/result` 채널링 완성 🎯

## 🏗️ 이동근 · 최태양 (인프라) 🏰

- [ ] **[3/31 화] 도메인 구매 및 선정**: 서비스 적합성 vs 비용 최적 도메인 선택 (오늘 결전!) ✨🏆
- [ ] **Nginx Reverse Proxy & SSL 적용**: 구매한 도메인에 Let's Encrypt HTTPS(SSL) 보안 인증서 설치 🔐

- [ ] **Docker Log Rotation**: 디스크 고갈 방지를 위한 10MB 기준 로그 순환 설정 적용 🐳

## 🏁 Milestone 2.0 성공 기준 (Blitz Mode)

1. [ ] **End-to-End**: 무임승차 동작 감지 -> 자동 클립 생성 -> API 전송 -> 대시보드 타임라인 즉시 표출.
2. [ ] **Security**: HTTPS 보안 연결과 사원번호 기반의 철저한 인증 체계 작동.
3. [ ] **Exam-Ready**: 사령관들이 시험 공부에 전념하는 동안 시스템이 문제없이 24시간 가동됨.
